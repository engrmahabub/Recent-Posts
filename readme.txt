== Description ==
**SM Recent Post** is a wordpress widget plugin to display **Recent Posts** in site sidebar.

= Features =
* Number of posts to show.
* For author page display only the author post or all recent posts. Default display all posts.

== Installation ==
1. Upload the entire `sm-recent-posts` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now SM Recent Post widget abailable in your widget section.

= Usage =

1. Go to Appearance > Widgets.
2. Enable SM Recent Post widget.
3. Enter widget title
4. Enter number of post you want to show;
5. Enter check options for author page or not.
8. Hit the save button.
